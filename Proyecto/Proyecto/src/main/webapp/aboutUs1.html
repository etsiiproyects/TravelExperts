Nuestro Proyecto

MusicExperts es un proyecto de 4 alumnos de la Escuela Tecnica Superior de Ingeniería Informática de Sevilla. Este proyecto consiste en realizar un mashup con diferentes APIs. Este grupo ha elegido hacer una pagina relacionada con la música. Usando TicketMaster para buscar conciertos, spotify con la que obtenemos las mejores canciones y se puede escuchar trozos de esas canciones.
